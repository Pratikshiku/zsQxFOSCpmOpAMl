## Urban Garbage Classification Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/726c3bd21d304f29be08a9ae79d6d7ab/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular Users, Administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Homepage
4. Forum
5. Announcements
6. Waste Information
7. Message Board
8. Policy Information
9. Personal Center
10. Waste Favorites


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Post Type Management
5. Announcement Type Management
6. Category Management
7. Policy Type Management
8. Forum Management
9. Announcement Management
10. Waste Management
11. Waste Message Management
12. Waste Favorites Management
13. Message Board Management
14. User Management
15. Policy Management
16. Banner Management
